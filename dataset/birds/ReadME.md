预期格式
```
┕━birds
    ┠─birds
        ┠─Black_footed_Albatross
        ┠─Crested_Auklet
        ┠─Groove_billed_Ani
        ┠─Indigo_Bunting
        ┗━Laysan_Albatross
    ┗━split_dataset.py
```
在`/dataset/birds`文件夹下启动`cmd`，输入以下命令
```cmd
python split_dataset.py --data_path=birds --split_rate=0.1
```

```
┕━birds
```
